# 1 Введение

Данный документ описывает план тестирования приложения ["GameShelf"](https://github.com/dispronesson/game-shelf). Документ предназначен для людей, 
выполняющих тестирование данного проекта. Цель тестирования - проверка соответствия реального поведения программы проекта и ее ожидаемого поведения, 
которое описано в [требованиях](../SRS.md).

# 2 Объекты тестирования

Функциональные возможности, подлежащие проверке:

- отображение списка игр на главной странице.
- бесконечная прокрутка и подгрузка игр.
- поиск игр по названию.
- переход на страницу игры.
- просмотр описания игры, характеристик, платформ и системных требований.
- просмотр скриншотов.
- динамическое обновление результатов без перезагрузки страницы.
